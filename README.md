# Madhav Ecommerce Sales Dashboard

## Overview
This project is an interactive Power BI dashboard designed to analyze ecommerce sales performance, profitability, customer behavior, and payment trends. The dashboard provides business insights through dynamic visualizations and filters.

## Tools Used
- Power BI
- Power Query
- Data Modeling
- DAX
- Data Visualization

## Key Metrics
- Total Sales Amount
- Total Quantity Sold
- Total Profit
- Average Order Value (AOV)

## Analysis Performed
- Sales by State
- Profit by Month
- Quantity by Product Category
- Profit by Sub-Category
- Customer-wise Sales Analysis
- Payment Mode Distribution
- Quarterly Performance Analysis

## Dashboard Features
- Interactive Quarter Filter
- State-wise Filtering
- KPI Cards for Business Performance
- Dynamic Charts and Visualizations
- Profitability Analysis
- Customer Contribution Analysis

## Key Insights
- Clothing contributed the highest share of product quantity.
- Cash on Delivery (COD) was the most preferred payment method.
- Profitability varied significantly across months.
- A small group of customers contributed a large portion of sales.
- Sub-category performance highlighted the most profitable product segments.
